Moodle theme afterburner
========================

This theme was part of the Moodle core distribution, but has now been moved to a separate repository.

Installation
------------

The best way to install this plugin is to get a copy from [moodle.org](https://moodle.org/plugins/view.php?plugin=theme_afterburner).
